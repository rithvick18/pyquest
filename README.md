# PyQuest - Immersive Python Learning

An interactive, AI-powered Python learning platform that brings coding education to life with gamification, real-time code execution, and personalized learning paths.

## Features

- **Interactive Curriculum**: Structured learning paths from beginner to advanced Python concepts
- **Real-time Code Execution**: Write and run Python code directly in the browser using Pyodide
- **AI-Powered Lessons**: Dynamic lesson content generated by Google Gemini AI
- **Context-Aware AI Assistant**: Get help tailored to your current lesson and progress
- **Gamification**: Earn XP, maintain streaks, and unlock achievements
- **Multiple Learning Paths**: 
  - Path 1: The Initiate (Foundations)
  - Path 2: The Apprentice (Data Structures & Loops)
  - Path 3: The Architect (Functions & Modules)
  - Path 4: The Master (Advanced Concepts)
- **Interactive Exercises**: Theory lessons, quizzes, and code challenges
- **Progress Tracking**: Visual roadmap showing completed and available lessons
- **Theme Customization**: Multiple light and dark themes
- **Search Functionality**: Quick lesson search with keyboard shortcuts (Ctrl+K)
- **Local Storage Persistence**: Progress, stats, and achievements saved locally

## Tech Stack

- **Frontend**: React 19 + TypeScript
- **Build Tool**: Vite 6
- **Code Editor**: Monaco Editor (@monaco-editor/react)
- **Python Runtime**: Pyodide (browser-based Python)
- **AI Integration**: Google Generative AI (@google/genai)
- **Markdown Rendering**: react-markdown
- **Styling**: CSS Variables for theming

## Run Locally

**Prerequisites:** Node.js 18+

1. Install dependencies:
   ```bash
   npm install
   ```

2. Set up environment variables:
   Create a `.env` file in the root directory and add your Gemini API key:
   ```
   VITE_GEMINI_API_KEY=your_api_key_here
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open your browser to `http://localhost:5173`

## Build for Production

```bash
npm run build
```

The optimized files will be in the `dist/` directory.

## Learning Paths

### Path 1: The Initiate (Foundations)
- First Steps: Hello World, Variables, Data Types
- String Mastery: String Operations, String Methods
- Logic & Control: Booleans, If Statements, Match Case

### Path 2: The Apprentice (Data Structures & Loops)
- Loops: For Loops, While Loops
- Lists: List Operations, List Methods
- Dictionaries: Dictionary Basics, Dictionary Methods

### Path 3: The Architect (Functions & Modules)
- Functions: Function Basics, Parameters, Return Values
- Modules: Importing Modules, Creating Modules
- Error Handling: Try/Except, Custom Exceptions

### Path 4: The Master (Advanced Concepts)
- OOP: Classes, Inheritance, Polymorphism
- File I/O: Reading Files, Writing Files
- Advanced Topics: Decorators, Generators, Context Managers

## Gamification System

- **XP System**: Earn XP based on lesson difficulty (Beginner: 50, Intermediate: 100, Advanced: 150)
- **Streaks**: Maintain daily learning streaks to unlock achievements
- **Achievements**: Unlock badges for milestones like first lesson, 10 lessons, perfect quizzes
- **Levels**: Progress through titles from Novice to Grandmaster (500 XP per level)

## Keyboard Shortcuts

- `Ctrl+K`: Open search modal
- `Esc`: Close modals

## Environment Variables

- `VITE_GEMINI_API_KEY`: Required for AI-powered lesson content and assistant
